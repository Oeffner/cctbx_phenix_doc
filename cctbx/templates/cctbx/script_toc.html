Example scripts

The scripts below illustrate how to use the cctbx.

Beginner

  • Generate a random model and compute its density map
    This example script shows how to generate a small centrosymmetric unit cell in P-1 with 15 atoms, and how to calculate phases and amplitudes to reconstruct the electron density.
  • Generate ideal helices and strands
    This script shows how to generate idealized helices and strands from a sequence.

Advanced

  • Compare secondary structure annotations numerically
    Several methods exist that determine the secondary structure of macromolecules. This script shows how to automatically determine secondary structure using several methods with cctbx tools and how to numerically compare the result.
